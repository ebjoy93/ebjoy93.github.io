Summary

The purpose of this project is to visualize the weather of 500+ cities across the world of varying distance from the equator. To accoplish this, we first pulled data from the OpenWeatherMap API to assemble a dataset.
After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included:

- temeprature
- cloudiness
- wind Speed
- humidity

This site provids the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
